# Music_Mixer
This repository includes the files for the Music Mixer assignment that is part of the Multimedia Authoring 2 class, which focuses on partners creating a fully functional music mixer or dj board using html, css, and javascript.
## Installation
No installation required. Simply download the zip file and extract.

## Usage
Open index.html in the browser of your choice.
